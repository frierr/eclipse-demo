# Eclipse Demo

A short demo of a top-down exploration game with some elements of puzzles and combat. Idea generated by ChatGPT.

## How to play

Play it online on [github pages](https://frierr.github.io/eclipse-demo/).

Controls:
* WASD - movement
* Mouse - rotation
* Mouse click - attack
* E - interaction
* Tab - inventory
